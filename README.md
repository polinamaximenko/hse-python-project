# Веб-приложение для автоматического ответа на вопросы по документам

### NLP-компонент (на выбор одна из реализаций поиска ответа на вопрос по тексту): 
FAISS

### API (возможные варианты для создания веб-платформы):
FastAPI
Streamlit
React

### Как используется нейросеть:
Используется предобученная модель для информационного поиска по векторному сходству или извлечения ответов
Документ разбивается на чанки (отрывки), которые векторизуются и сравниваются с вопросом пользователя

### Результат работы:
Пользователь загружает текстовый документ (PDF, DOCX) или вставляет текст в поле. Затем задает вопросы по содержимому документа. Система находит и выделяет в тексте точный ответ
